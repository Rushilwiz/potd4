This is a React/NextJS app that connects to my database hosted at [https://mysql.rushil.land](https://mysql.rushil.land).

Here are the steps to run the app locally.

First, import `schema.sql` to your server of choice and create a .env file based on .env.sample with the connection info.

Then, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
